# Financial-Aid-PKU

iOS project for Financial-Aid-PKU

## Setup Dev Env

```
brew install swiftlint
pod install
```

## Development

### Run linter and auto fix

Run lint and fix automatically
```
swiftlint
swiftlint autocorrect
```

### Deploy to staging/prod

```
Run Xcode release build
```
